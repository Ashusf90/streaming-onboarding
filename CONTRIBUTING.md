# Contributing to Streaming Onboarding

Thank you for helping keep this resource up to date. Contributions are welcome — whether you are adding a new link, reporting a broken one, or fixing outdated information.

## How to add a resource

1. Fork this repository and create a branch from `main`.
2. Add your link to the most relevant section in `README.md`.
3. Use the existing format:
   ```
   - [Title:](URL) Brief description of what the resource covers.
   ```
4. Prefer links to resources that are:
   - Accurate and technically sound
   - From a reputable source (major streaming company, standards body, or well-known practitioner)
   - Stable — not behind a paywall or likely to disappear
5. Open a pull request with a short description of what you added and why it is useful.

## How to report a broken or outdated link

Open an issue with:
- The broken URL
- The section it appears in
- A suggested replacement URL if you have one

## How to fix a broken link yourself

Follow the same steps as adding a resource, but replace or remove the broken link. If you are removing a link with no replacement, briefly note why in the pull request description (e.g. "site is down", "company shut down").

## Format guidelines

- Keep descriptions concise — one sentence is enough.
- Do not add affiliate links or sponsored content.
- Sections are roughly ordered from foundational to advanced — add new links in a position that fits the learning progression.
